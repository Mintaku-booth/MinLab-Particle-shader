# MinLab Particle Shader

MinLABのパーティクル用に作成したカスタムシェーダーです。

## 導入方法

### 必要なリポジトリの登録

VCCに以下のリポジトリを順番に登録してください。

**1. z3y's packages（Shader Graph VRC）**
https://z3y.github.io/vpm-packages/index.json

**2. AudioLink**
（VCCのCuratedリポジトリに含まれているため追加不要）

**3. MinLab Particle Shader**
https://mintaku-booth.github.io/MinLab-Particle-shader/index.json

### プロジェクトへの追加

1. VCCでプロジェクトを開く
2. `Manage Packages` から `MinLab Particle Shader` を探す
3. `+` ボタンでインストール

## 動作環境

- Unity 2022.3
- VRChat SDK 3.x
